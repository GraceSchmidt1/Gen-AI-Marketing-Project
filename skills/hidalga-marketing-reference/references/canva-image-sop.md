# Canva Image Creation SOP

*Source: Canva_Image_Creation_SOP.docx — Author: Grace Schmidt, 12/12/2025*

---

## Purpose
Standard process for creating social media images in Canva for Instagram and LinkedIn using the Hidalga brand system.

---

## Accessing Canva

- Open the **Canva Pro team account**: [canva.com](https://www.canva.com/)
- Navigate to **Projects** in the left sidebar
- Open the **Social Media Assets** folder
- Select the appropriate platform folder (LinkedIn or Instagram)

**Folder path:** Canva → [Projects](https://www.canva.com/projects) → [Social Media Assets](https://www.canva.com/folder/FAF2YEajqTI) → LinkedIn / Instagram folder

---

## Instagram Image Creation

### Pre-Work Requirements
- A draft of the caption must be completed before creating the image (normally already in Planner at draft stage)
- The draft caption guides both the **visual direction** and the **on-image text**
- Image source will be one of: team photo (SharePoint), Unsplash image, Canva image, or provided media kit image

### Template Setup
- Open the designated Instagram image template (Instagram Post 4:5 size)
- Template link: [Instagram Templates Folder](https://www.canva.com/folder/FAF3rKMJhL8)
- Populate the image text box with a **condensed version of the caption — limited to two sentences**
- AI tools may be used to assist with condensing if needed
- **First sentence: bold. Second sentence: not bold.**
- Complete the image draft without the custom photo first (get the template and text right before adding photo)

### Photo Selection & Sourcing

**Option 1 — SharePoint (team/event photos)**
- Photos taken at events or provided by team members
- Download as PNG, upload into Canva
- Link: [SharePoint Media Folder](https://hidalga.sharepoint.com/sites/Docs/HHT%20600%20%20Sales%20%20Marketing/Forms/AllItems.aspx?id=%2Fsites%2FDocs%2FHHT%20600%20%20Sales%20%20Marketing%2FHHT%20605%20%2D%20Marketing%2FHHT%20605%2E03%20%2D%20Media)

**Option 2 — Unsplash (free stock images)**
- Download as PNG, upload into Canva
- Record the image link (available during download) for crediting purposes
- Link: [unsplash.com](https://unsplash.com/)

**Option 3 — Canva native images**
- Use Canva's built-in image library

**Option 4 — Media kit images**
- Provided externally; will already be in Planner

> Download all images at their automatic size as PNG before uploading into Canva.
