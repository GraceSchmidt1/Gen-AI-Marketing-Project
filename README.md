# Hidalga Marketing Dashboard

Interactive social media analytics dashboard for Hidalga Technologies. Tracks performance across Facebook, Instagram, and LinkedIn with weekly ICP-segment reviews, UTM standards, and AI-generated insights.

## Quick Start

```bash
# 1. Clone the repo
git clone <repo-url>
cd Hidalga-Marketing-Dashboard

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run
streamlit run app.py
```

The dashboard opens at `http://localhost:8501`.

## Features

### 📊 Overview Tab
- KPI cards — total impressions, engagement, clicks, posts published
- Impressions over time by platform (line chart)
- Engagement and format breakdown charts
- Month-over-month comparison with % delta
- Top performing posts table — sortable by any metric, color-coded by platform and ICP segment
- AI-generated insights via Claude Opus (requires API key)
- Export to CSV or multi-sheet Excel

### 📅 Weekly Review Tab
Weekly cadence: review every Monday for the prior week.

- **Segment performance table** — impressions, clicks, CTR, key actions, and conversions broken out by ICP segment
- **Automatic flags**: 🟢 Overperformer (CTR ≥ 1.5× avg) · 🔴 Underperformer (CTR ≤ 0.5× avg) · ⚠️ Low reach
- **Post drill-down** — every post for the selected week, filterable by segment, with per-post CTR flags
- **Discussion notes** — record observations and decisions; saves to `data/weekly_notes.json`; download as a formatted `.txt` summary
- **Discussion checklist** — standard questions to walk through each Monday

> Note: Instagram does not export link clicks via Loomly, so CTR is calculated from Facebook + LinkedIn only.

### 🔗 UTM Standards Tab
- Full naming convention reference for all five UTM parameters
- ICP short codes for `utm_content` tagging (`enterprise`, `technical`, `smb`, `awareness`)
- Example tagged URLs per platform
- **Live UTM builder** — fill in the form, get a tagged URL instantly, download parameters as CSV
- GA4 implementation guide — custom dimensions, attribution queries, and a path to API-connect conversions

## Data

Data is exported from Loomly and placed in the `data/` folder.

| File | Description |
|---|---|
| `data/Facebook-data(data-cleaned).csv` | Facebook post metrics |
| `data/Instagram-data(data-cleaned).csv` | Instagram post metrics |
| `data/linkedin-data(data-cleaned).csv` | LinkedIn post metrics |
| `data/icp_segments.csv` | ICP segment definitions — edit to match Hidalga's ICPs |
| `data/post_segments.csv` | Maps each post to an ICP segment; add conversions from GA4 weekly |
| `data/weekly_notes.json` | Auto-created on first save; stores weekly discussion notes |

### Updating data

1. Export new CSVs from Loomly (same column format)
2. Replace the files in `data/`
3. Restart the app — `st.cache_data` will reload automatically

### Updating ICP segment assignments

Open `data/post_segments.csv` and edit the `ICP_Segment` column to match the actual intent of each post. Valid segment names must match those in `data/icp_segments.csv`. Fill in the `Conversions` column each Monday from GA4.

```csv
Date,Platform,ICP_Segment,Conversions
2026-03-24,LinkedIn,Enterprise Decision Makers,3
2026-03-24,Instagram,Brand Awareness,0
```

## AI Insights

Insights are generated by **Claude Opus 4.6** and triggered on demand (sidebar button). Provide your API key one of two ways:

**Option 1 — Sidebar input:** paste the key directly in the app each session.

**Option 2 — Environment variable (recommended):**
```bash
export ANTHROPIC_API_KEY=sk-ant-...
streamlit run app.py
```

Copy `.env.example` to `.env` and fill in the key if using a tool like `python-dotenv`.

Get an API key at [console.anthropic.com](https://console.anthropic.com).

## UTM Naming Convention

| Parameter | Format | Example |
|---|---|---|
| `utm_source` | platform (lowercase) | `linkedin` |
| `utm_medium` | `organic-social` or `paid-social` | `organic-social` |
| `utm_campaign` | `[year]-[quarter]-[initiative]` | `2026-q2-product-launch` |
| `utm_content` | `[format]-[icp-short]` | `image-enterprise` |
| `utm_term` | topic keyword (optional) | `ai-solutions` |

Use the UTM Builder tab to generate tagged URLs without typing manually.

## Weekly Review Process

1. **Every Monday** — open the Weekly Review tab and select the prior week
2. Review the segment performance table — note any 🟢 or 🔴 flags
3. Walk through the discussion checklist (expandable in the tab)
4. Record key observations and next steps in the notes field, click **Save Notes**
5. Update `data/post_segments.csv` with GA4 conversion counts for the week
6. Click **Download Summary** to save a `.txt` record for the team
